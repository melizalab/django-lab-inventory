- **Setup Time**: 1 minute

- **Best For**: Quick demo, testing

### 3. Gitpod
- **URL**: https://gitpod.io/#https://github.com/melizalab/django-lab-inventory
- **Setup Time**: 2 minutes
- **Cost**: Free (50 hours/month)
- **Requirements**: GitHub account
- **Best For**: Cloud development

**Steps**:
1. Click the link or "Open in Gitpod" badge
2. Sign in with GitHub
3. Automatic setup completes
4. Browser opens automatically

### 4. Google Cloud Shell
- **URL**: https://shell.cloud.google.com/
